# AmbientLightLevel Enumeration (Android only)

The ambient light level around the device.

>```java
> public enum AmbientLightLevel {
>   LIGHT,
>   DARK
> }
>```

## Values

### Light

The device is in a bright environment.

### Dark

The device is in a dark environment.
